\begin{abstract}
 Este documento continene un analisis del conjunto de datos denominado "Precios de Combustibles" el cual fue obtenido del repositorio de datos abiertos de Colombia - www.datos.gov.co. Este analisis fue realizado con el fin de obtener informacion de las variaciones en los precios de los combustibles de acuerdo a diferentes variables en una linea de tiempo.
\end{abstract}
